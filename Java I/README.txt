Это комментарий к тому "Java I".

Использование ссылок на объекты происходит строго как описано, так как принятаяв языке Java строгая типизация не позволяет использовать ссылки на объекты так же фривольно, как в JavaScript.
Тема конструкторов раскрыта сполна, и ее материал следует понимать как он и прописан - буквально. Эта тема может требовать лишь следующих комментариев:
код конструктора предназначен для инициализации членов объектов пользовательскими значениями, поэтому, он должен содержать лишь вышеобозначенные операции (и не должен производить работу с системными данными), поэтому же, реально существующие конструкторы не сложнее, чем примеры из учебных пособий;
вызов конструктора суперкласса в конструкторе подкласса необходим лишь когда это задумано программистом (как и оговорено, общие с суперклассом члены могут быть инициализированы и самим конструктором подкласса);

тема конструкторов не требует прочих комментариев.
